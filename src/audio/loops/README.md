# 环境音循环文件

请将你的音频文件放到此目录。

## 文件命名规范

### 雨声场景 (rain)
- `rain_1.mp3` - 基础雨声
- `rain_2.mp3` - 雨声变体
- `rain_birds.mp3` - 带鸟声的雨

### 咖啡馆场景 (cafe)
- `cafe_1.mp3` - 咖啡馆环境音
- `cafe_2.mp3` - 咖啡馆变体

### 风声场景 (wind)
- `wind_1.mp3` - 基础风声
- `wind_2.mp3` - 风声变体

## 支持格式
MP3, WAV, OGG, M4A

## 工作原理
- 引擎会自动加载此目录中的文件
- 多个文件时随机选择下一个播放
- 文件间有平滑的交叉淡化过渡
