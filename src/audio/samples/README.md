# 音效采样下载指南

请从以下网站下载免费音效，放到 `src/audio/samples/` 目录：

## 推荐网站

- [Pixabay Sound Effects](https://pixabay.com/sound-effects/) - 免费，无需署名
- [Freesound.org](https://freesound.org/) - CC0 公共领域

## 需要的文件

| 文件名 | 关键词搜索 | 建议时长 |
|--------|-----------|---------|
| `raindrop_1.mp3` | raindrop, water drop | < 1秒 |
| `raindrop_2.mp3` | raindrop, drip | < 1秒 |
| `raindrop_3.mp3` | rain drop | < 1秒 |
| `cup_clink_1.mp3` | cup clink, glass | < 1秒 |
| `cup_clink_2.mp3` | ceramic, dish | < 1秒 |
| `spoon_stir.mp3` | spoon, stir, coffee | 1-2秒 |

## 下载提示

1. 选择**短小**的单次音效，不要选循环音
2. 优先选择 MP3 格式
3. 下载后重命名为上表中的文件名

引擎会自动检测并使用这些采样。如果采样不存在，会回退到合成音效。
